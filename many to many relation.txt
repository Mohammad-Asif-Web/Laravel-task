jdi amra posts and tag niye chinta kri, akti tag onek gulo poster hote pare. abar akti post onek gulo tag er hote pare. akhon databaser same table e akti post er jnne multiple tag, ba akti tag er jnne multiple post rakha jay na. tai ader jnne jnne alada table banate hobe. ar ai table kei bola hoy pivot table.

suppose: post id no.1 e tag id ache = 1, 2, 3
tag_id no. 1 e post-id ache = 5, 6, 7
ai duiti column er pivot table jemon hobe:

post_id		tag_id
   1		1
   1		2
   1		3
   5		1
   6		1
   7		1

laravel e pivot table create er kisu rule ache.
1) table duitir nam singular hote hobe
2) alphabetical dui table er modde jar alphabet age porbe tar nam age hobe. ex:
post ar tag er moddhe 'p' okkhor 't' okkhorer age hoy. tai table name hobe--
'post_tag', jdi post ar category relation hoy, tahle eder pivot table er name hoto
'category_post'.

akhon post_tag table create krte command:
-> php artisan make:migration create_post_tag_pivot_table --create=post_tag
